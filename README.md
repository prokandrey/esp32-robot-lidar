# ESP32 - Подключение компонентов

## Полная таблица пинов ESP32

| Компонент | Сигнал | Пин ESP32 |
|-----------|--------|-----------|
| **A4988 #1 (левый двигатель)** | STEP | GPIO19 |
| | DIR | GPIO18 |
| | ENABLE | GPIO17 |
| **A4988 #2 (правый двигатель)** | STEP | GPIO23 |
| | DIR | GPIO22 |
| | ENABLE | GPIO21 |
| **Лидар** | TX (на вход ESP32) | GPIO16 |
| **Кнопка ROUTE** | INPUT_PULLUP | GPIO26 |
| **Кнопка START** | INPUT_PULLUP | GPIO27 |
| **Серво 0** | PWM | GPIO33 |
| **Серво 1** | PWM | GPIO25 |

## Подключение драйвера A4988

| Пин A4988 | Куда подключать |
|-----------|-----------------|
| STEP | GPIO19 (мотор 1) / GPIO23 (мотор 2) |
| DIR | GPIO18 (мотор 1) / GPIO22 (мотор 2) |
| EN | GPIO17 (мотор 1) / GPIO21 (мотор 2) |
| VDD | 3.3V ESP32 |
| VMOT | 12-24V внешний БП |
| GND | GND ESP32 + GND внешнего БП |
| MS1 | GND |
| MS2 | 3.3V |
| MS3 | GND |

## Подключение лидара

| Лидар | ESP32 |
|-------|-------|
| TX | GPIO16 |
| VCC | 5V |
| GND | GND |

## Подключение кнопок

| Кнопка | ESP32 | Примечание |
|--------|-------|------------|
| ROUTE | GPIO26 | INPUT_PULLUP, активен LOW |
| START | GPIO27 | INPUT_PULLUP, активен LOW |

## Подключение сервоприводов

| Серво | ESP32 | Питание |
|-------|-------|---------|
| Серво 0 | GPIO33 | 5V |
| Серво 1 | GPIO25 | 5V |
